# X-Puzzle-Kit AI Agent Instructions

> [!IMPORTANT]
> 这是本项目的行为最高准则。AI 助理在执行任何任务前，**必须**首先阅读并遵守本文件中的规则。

## 1. 国际化 (i18n) 全量覆盖规则

- **全量对齐铁律**：任何 UI 字符串的增删改，**必须在同一个任务/提交中同步更新全部 7 种语言包** (`zh_CN`, `en`, `zh_TW`, `ja`, `ko`, `es`, `fr`)。
- **严禁采样**：严禁仅更新部分语言后声称“已完成全量更新”。交付前必须逐一核对。
- **键位一致性**：所有语言包的 Key 必须完全对等。

## 2. 行为原子性与自检

- **交付前核查**：在通过 `notify_user` 声明完成前，必须手动或通过脚本运行以下核查：
  1. `npm run build:web`：确保构建链路通畅。
  2. `npm run lint`：确保 JSON 无语法错误及重复键。
  3. **多语言对等性**：对比 7 个语言包的键位总数，确保无漏项。
- **记忆保障**：本项目根目录下的 `.agent/instructions.md` 是 AI 助理执行任务的首要依据。每次任务开始必须回顾此文件，严禁依赖猜测或碎片记忆。

---

_这些准则是为了将您的提醒转化为本机的物理约束，减少不必要的沟通磨损。_

在通过 `notify_user` 声明任务完成前，**必须**执行以下自检：

1. [ ] **全量编译**：运行 `npm run build:web` 确保最终包可产出。
2. [ ] **多语言核对**：检查所有 7 个语言包是否已同步最新词条。

---

_这些准则是为了减少重复沟通、提升交付质量而设立的。请严格执行，不要让用户分心于这些琐事。_
