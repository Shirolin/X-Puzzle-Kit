# X-Puzzle-Kit iOS 快捷指令逻辑说明

为了实现从照片 App 一键导入图片到 Web 版，快捷指令需要遵循以下逻辑设计：

## 核心流程

1.  **输入接收**：设定快捷指令为“在共享表单中显示”，输入类型限制为“图像”。
2.  **图片处理**：
    - 获取输入中的所有图像。
    - 循环处理每一张图片。
3.  **URL 编码/拼接**：
    - 由于 Web 环境安全限制，不建议通过 URL 传递大量 Base64（URL 长度限制）。
    - **推荐方案 A (剪贴板)**：
      - 将选中的图片合并或依次拷贝到剪贴板。
      - 打开 X-Puzzle-Kit 网页版。
      - 网页版监听 `paste` 事件（App.tsx 已实现拖拽和点击上传，可进一步完善剪贴板直接注入）。
    - **推荐方案 B (快捷指令直接打开)**：
      - 打开 URL: `https://your-domain.com/`。
      - 用户在网页点击“从剪贴板粘贴”。

## 快捷指令构建步骤 (手动参考)

1.  **接受输入**：`Shortcut Input` (Images)。
2.  **动作 1**：`Copy Shortcut Input to Clipboard`。
3.  **动作 2**：`URL` -> `https://x-puzzle.pages.dev` (待定域名)。
4.  **动作 3**：`Open URL`。

## 开发者建议

未来可以实现在 `index.html` 中通过 Web Share Target API 接收图片，但这需要 PWA 环境且目前的浏览器支持度不一。
目前的“拷贝至剪贴板 + 唤起网页”是最稳定的跨设备方案。
